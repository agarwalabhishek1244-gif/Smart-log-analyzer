# Smart Log Analyzer with AI Insights

This project analyzes system logs using Python and generates insights using AI.

## Features:
- Log parsing and analysis using Pandas
- Filtering and grouping log data
- Identifying error patterns
- AI-based insights using OpenAI API

## Tech Stack:
- Python
- Pandas
- OpenAI API

## How it works:
- Reads log data from CSV
- Processes and filters logs
- Sends log messages to AI model
- Generates summary insights

## Setup:
- Install dependencies: pip install -r requirements.txt
- Set your OpenAI API key as environment variable
